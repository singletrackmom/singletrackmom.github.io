Resume 4th Draft

Module 2 — Resume

Description

Your resume content should be strong by now. This draft focuses on visual polish, typographic precision, and making it completely error-free — ready for employer submission.

Learning Objectives

By completing this assignment you will be able to:

  • Apply typographic best practices to a professional document
  • Identify and correct all formatting inconsistencies
  • Produce a near-final resume ready for employer submission

Steps

  1. Review all previous instructor feedback.
  2. Focus on design this round: consistent type sizes, tabs set correctly (not using the space bar), even margins, clear visual hierarchy.
  3. Proofread for spelling, grammar, and punctuation. Read it backward — bottom to top — to catch typos your eye normally skips.
  4. Optional but strongly encouraged: schedule a review with Career Services for extra credit (see Extra Credit module).
  5. Save as PDF and submit here.

Example

At this stage look for:

  • Section headers all the same size and weight?
  • Spacing between sections consistent?
  • Bullet indents identical throughout?
  • Dates formatted the same every time (May 2024, not May '24 in one place and 05/24 in another)?

How to Submit
