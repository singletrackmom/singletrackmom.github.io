Resume 2nd Draft

Module 2 — Resume

Description

Your instructor has reviewed your first draft and left feedback. Now apply that feedback and improve the content, structure, and clarity of your resume.

Learning Objectives

By completing this assignment you will be able to:

  • Incorporate instructor feedback from the first draft
  • Strengthen action verb language throughout
  • Ensure all content is accurate, relevant, and clearly formatted

Steps

  1. Review the feedback your instructor left on your first draft.
  2. Make all suggested changes. If you disagree with a suggestion, note it in the comments — but give a reason.
  3. Check every bullet: does it start with a strong action verb? Is it specific? Does it describe a result, not just a task?
  4. Review spelling, punctuation, capitalization, and consistent formatting throughout.
  5. Aim for one page. Tighten language rather than shrinking font below 8pt.
  6. Save as PDF and upload here.

Example

At this stage focus on:

  • Every bullet starting with a past-tense action verb
  • Quantifying results where possible: "Designed 12 social media assets for a nonprofit campaign" beats "Made social media posts"
  • No typos — none
