Midterm Portfolio Preparation Plan

Module 3 — Cover Letter, Elevator Pitch & Midterm Prep

Description

Your portfolio is the centerpiece of your job search. Before you can present it at midterm, you need a clear plan: which projects will you show, what needs to be improved, and when will you do the work?

Learning Objectives

By completing this assignment you will be able to:

  • Identify your strongest existing projects and assess what needs improvement
  • Write professional project descriptions covering goals, process, and outcome
  • Create a realistic week-by-week production schedule

Steps

  1. Go through all the work you have created in your degree. Select at least 3 projects to highlight at midterm — your best work or work you plan to improve.
  2. For each project write a description covering: the goals of the project, your role, the process you used, and the outcome (3-5 sentences each).
  3. List everything that needs to be fixed or updated in each project before midterm.
  4. Create a week-by-week schedule from now until midterm showing what you will work on each week. Be specific: project name and task (e.g., 'Week 3 — Fix kerning on poster title, re-export at 300dpi').
  5. Decide where you will host your portfolio: Behance, Adobe Portfolio, ArtStation, Vimeo/YouTube for reels, or a custom site.
  6. Save as PDF and submit here.

Example

This assignment works well with AI. List each project and the tasks needed, then ask Claude or ChatGPT to build a production schedule for you. A strong project description looks like this:

"Brand identity for a fictional sustainable clothing company called Verdant. Goals were to create a cohesive visual system that felt modern, earthy, and premium. I developed the logo, color palette, type system, packaging mockups, and brand standards guide."
